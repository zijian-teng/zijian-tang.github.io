C++ Primer 复习笔记 - 查漏补缺

发表于2021-10-07|更新于2021-10-07|C++

  • 1 开始

    • 【P4】查看 main 函数返回值
      • UNIX: echo $?
      • Windows: echo %ERRORLEVEL%
    • 【1.2】流：随时间顺序生成/消耗的字符序列
      • << 运算符接受两个运算对象，左侧为 ostream，将右侧对象写入左侧 ostream，返回左侧 ostream
    • 【1.4.3】if(stream) 检测流的状态
      • 未遇到错误返回 true
      • 无效的流（如 EOF，无效输入等）返回 false
      • EOF
        • UNIX：Ctrl + D
        • Windows：Ctrl + Z，Enter
    • 【1.5】标准库头文件没有后缀
    • 【P19】引用标准库的头文件用尖括号 #include <vector>，其他头文件用 #incldue "header.h"
      • 重定向：a.out < inputFile.txt > outputFile.txt
    • 【P20】方法 method 即类的成员函数 member function
    • 【小结 P23】流的刷新
      • 默认读 cin 会刷新 cout
      • 程序非正常终止也会刷新 cout
      • endl/flush 等操纵符强制刷新 cout
      • cerr 不缓冲
    • 表达式：一个或多个运算对象，一个或多个运算符
      • i+j
      • ++i
    • 语句：表达式 + 分号 ;

2 变量和基本类型

  • 【P29】数据结构：数据以及其所支持的操作

  • 基本内置类型

    • 算数
      • 浮点：float，double等
      • 整型：int，char，bool 等
    • 空类型 void

  • C++ 只规定最小尺寸/最小有效位数，编译器可以选择更大尺寸/更高有效位数

    • C++ 没有规定有符号数如何表示，但约定正负应尽量平衡
    • char 具体是有符号还是无符号的由编译器决定

  • 【P32】如何选择类型

    • 明确知道不可能是负数：uint
    • 一般算数运算：int，double

  • 【P34】int 和 uint 一起运算，int 会自动转为 uint ！

    • int a = -1;
uint b = 1;
cout << a * b << endl; // 不会输出 -1！

uint u1 = 1;
uint u2 = 2;
int i = -3;
cout << u1 - u2 << endl; // 不会输出 -1！
cout << i + u2 << endl; // 不会输出 -1！

  • 【P24】字符串常量：双引号包围的 ”0..N 字符序列“

  • 字面值类型 【P59】

    • nullptr 是字面值 【P48】
    • 字面值常量【P35】【P267】
      • 需要转义的字符
        • 不可打印
        • 特殊含义：，?，\，"
