条条大路走不通之 Linux 源码方式安装

安装序列化库 CDR

如果最后一步提示没有权限，sudo 一下即可

安装 foonathan 分配器

make 过程中还会从 github 下载代码，如有必要，用 sed 命令把 CMake 生成的临时文件 build/foo_mem-ext-prefix/tmp/foo_mem-ext-gitclone.cmake 中 github.com 替换成 hub.fastgit.org

安装 FastDDS

安装 Fast DDS Gen（IDL 生成 C++ 代码）

上述步骤中存在诸多 gradle 的坑：

🕳 5：系统 gradle 版本和 FastDDS 的不匹配

使用文件夹下 ./gradlew assemble，自动下载和当前 FastDDS 匹配的 gradle 版本

🕳 6：gradlew 可能因代理问题无法下载 gradle 包

  1. 手动下载 gradle 包，放到 ~/Fast-DDS/src/fastddsgen/thirdparty/idl-parser/gradle/wrapper
  2. 编辑 fastddsgen/thirdparty/idl-parser/gradle/wrapper/gradle-wrapper.properties，删除 distributionUrl 字段的 https://services.gradle.org/distributions/
  3. 运行 ./gradlew assemble，编译之后的程序在 ~/Fast-DDS-Gen/scripts/fastddsgen 目录
  4. 将该目录添加到 PATH 路径
